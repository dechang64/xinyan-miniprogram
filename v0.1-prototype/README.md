# 心颜 v0.1 — Streamlit 原型

> **照镜子, 也是为了更好的自己**

心颜是一款微信小程序 v0.1 原型, 用 **Streamlit** 跑通 3 个核心 page, 验证产品哲学:
- **每日一经** (易经 / 清静经 / 道德经 / 黄帝内经)
- **每日一汤** (9 体质 + 4 季 + 30 款养生美颜汤)
- **共修堂** (3 任务打卡 + 心愿流 + 镜中自评)

## 严守 6 条原则 (user 拍板)

1. **与医疗隔离** — 不说治疗/改善/缓解, 用滋养/涵养/陪伴/共修
2. **与祺臻心理独立** — 完全独立 Streamlit app, 不挂「祺臻」品牌
3. **照镜子也是为了更好的自己** — 启动页 + 镜中 page + 品牌副标
4. **重点社群** — 共修堂是核心壁垒, 严守红线 (不评美丑 / 不卖货 / 不医疗)
5. **每日一经** — 30 篇预生成, 6 海报模板, 朋友圈分享
6. **每日一汤** — 9 体质 (王琦) × 4 季 = 30 款, 简版做法

## 跑起来

```bash
cd xinyan_prototype
pip install -r requirements.txt
streamlit run app.py
```

浏览器打开 `http://localhost:8501`

## v0.1 不做什么 (为 v0.5 留口子)
