# 通用规范 (5 段统一)

- 采样率 44100 Hz / 16-bit
- 立体声 (Stereo, BMLD 立体感)
- 声道时间差 ≤ 30 μs (JNΔT 严守)
- 音量 25-60 dB SPL (T/CRHA036—2024 §7.3.4)
- 时长 5-10 分钟
- 段间留白 ≥ 200ms (防 Zwicker Tone)
- 主旋律 ≤ 1.5 kHz (保留 BMLD 立体感)
- 谐波数 5-10 (大三和弦天然)

# 5 音统一 A4-E5 (440-659 Hz)

- 全部 < 1.5 kHz BMLD 区
- 保留 BMLD 立体感 (Fastl §15)

# 共修堂空间规范 (T/CRHA036—2024 §5)

- 房间 12-20 m²
- 温度 18-22℃
- 湿度 50-60%
- RT60 ≤ 0.5 秒

# 评估规范

- ACR 5 等级量表 (优/良/中/差/劣)
- 最少 16 名体验者
- 6 周以上 (Music Therapy Meta 2020 SMD=-1.33 有效阈值)
