# 心颜 (XINYAN) — 微信小程序 + Streamlit 原型

> **照镜子, 也是为了更好的自己**

心颜是一款关注现代女性「身心滋养」的陪伴型应用。
技术栈: **微信小程序 + 微信云开发 + CloudBase AI / AMAX** (与祺臻心理共用底座, 但**完全独立**品牌)

## v0.1 原型 — 跑起来

```bash
cd v0.1-prototype
pip install -r requirements.txt
streamlit run app.py
```

浏览器打开 `http://localhost:8501`

### 3 个核心 page

1. **每日一经** (`pages/1_每日一经.py`) — 30 篇经文 (易经/清静经/道德经/黄帝内经) + 简释 + 6 海报模板
2. **每日一汤** (`pages/2_每日一汤.py`) — 9 体质 (王琦) + 30 款养生美颜汤 + 换一换
3. **共修堂** (`pages/3_共修堂.py`) — 3 任务打卡 + 镜中自评 + 心愿流

## 严守 6 条原则 (user 拍板)

1. **与医疗隔离** — 不说「治疗/改善/缓解」, 用「滋养/涵养/陪伴/共修」
2. **与祺臻心理独立** — 完全独立品牌, 浅米白主题 (vs 祺臻深色)
3. **照镜子也是为了更好的自己** — 启动页 + 镜中 page + 品牌副标
4. **重点社群** — 共修堂是核心壁垒, 严守红线 (不评美丑/不卖货/不医疗)
5. **每日一经** — 30 篇预生成, 6 海报模板, 朋友圈分享
6. **每日一汤** — 9 体质 × 4 季 = 30 款, 简版做法
